# frontend-starter-kit

This is minimum stack project for development with TypeScript.

## Usage
```sh
$ npm run dev
// open http://localhost:8080 in web browser

$ npm run build
// generate assets into dist dir
```

## Stack
- TypeScript
- Webpack
